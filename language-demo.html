🌍 多语言功能使用指南

📋 功能特点

  • 四种语言支持：中文、英文、西班牙文、法文
  • 实时切换：无需刷新页面即可切换语言
  • 本地存储：记住用户的语言偏好
  • 课件多语言：每个课件都支持多语言标题和描述
  • 完整界面翻译：导航、按钮、提示信息全部翻译

🔧 API 使用方法

1. 切换语言

// 切换到英文
CoursewareManager.setLanguage('en');

// 切换到西班牙文
CoursewareManager.setLanguage('es');

// 切换到法文
CoursewareManager.setLanguage('fr');

// 切换回中文
CoursewareManager.setLanguage('zh');

2. 获取当前语言

const currentLang = CoursewareManager.getCurrentLanguage();
console.log('当前语言:', currentLang);

3. 获取翻译文本

// 获取特定翻译
const siteTitle = CoursewareManager.getTranslation('site.title');
const navAll = CoursewareManager.getTranslation('nav.all');

// 或者使用简短方法
const loadingText = CoursewareManager.I18n.t('courseware.loading');

4. 添加多语言课件

const newCourseware = {
  title: {
    zh: "新课件标题",
    en: "New Courseware Title",
    es: "Título del Nuevo Material",
    fr: "Titre du Nouveau Support"
  },
  description: {
    zh: "课件描述",
    en: "Courseware description",
    es: "Descripción del material",
    fr: "Description du support"
  },
  category: "science",
  embedUrl: "https://example.com/embed",
  icon: "🔬"
};

CoursewareManager.addCourseware(newCourseware);

🎯 使用建议

  • 语言切换器：位于页面右上角，用户可以随时切换语言
  • 自动保存：用户选择的语言会自动保存到本地存储
  • 回退机制：如果某个语言的翻译缺失，会自动回退到中文
  • 兼容性：支持旧格式的课件数据（纯字符串标题和描述）

📝 添加新语言

要添加新语言，需要在 script.js 文件中的 translations 对象中添加相应的翻译数据：

const translations = {
  // 现有语言...
  de: { // 德语
    site: {
      title: "Bildungsplattform für Kursmaterialien"
    },
    nav: {
      all: "Alle Kurse",
      // 其他翻译...
    }
  }
};

然后在HTML中的语言选择器中添加新选项：

<option value="de">Deutsch</option>
